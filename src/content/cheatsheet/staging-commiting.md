### `git status`  
See changes, untracked files, and current branch.

### `git add <file>`  
Stage a file for the next commit.

### `git commit -m "message"`  
Commit staged changes with a message.

### `git commit --amend`  
Edit the last commit (e.g., fix message or add files).
